# Template Development

This section provides tutorials for creating and publishing CyanPrint templates. Templates are the blueprints that users use to generate new projects.

## Available Tutorials

- [Create a Template (JavaScript)](/docs/developer/tutorial/template/javascript) - Create a template using JavaScript
- [Create a Template (Python)](/docs/developer/tutorial/template/python) - Create a template using Python
- [Create a Template (C#)](/docs/developer/tutorial/template/csharp) - Create a template using C#
- [Publish Your Template](/docs/developer/tutorial/template/publish) - Publish your template to the CyanPrint registry

Each tutorial provides detailed instructions with examples to help you understand how to develop templates for CyanPrint.
