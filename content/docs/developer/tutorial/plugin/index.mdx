# Plugin Development

This section provides tutorials for creating and publishing CyanPrint plugins. Plugins extend CyanPrint's capabilities by performing additional actions after project generation.

## Available Tutorials

- [Create a Plugin (JavaScript)](/docs/developer/tutorial/plugin/javascript) - Create a plugin using JavaScript
- [Create a Plugin (Python)](/docs/developer/tutorial/plugin/python) - Create a plugin using Python
- [Create a Plugin (C#)](/docs/developer/tutorial/plugin/csharp) - Create a plugin using C#
- [Publish Your Plugin](/docs/developer/tutorial/plugin/publish) - Publish your plugin to the CyanPrint registry

Each tutorial provides detailed instructions with examples to help you understand how to develop plugins for CyanPrint.
