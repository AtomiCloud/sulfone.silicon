# Introduction

CyanPrint is a next-generation language-agnostic templating platform that helps you create projects from templates. It provides a powerful alternative to traditional project scaffolding tools with features like customizable templates, plugin support, and a central registry.

## Quick Start

Quickly get started with CyanPrint by following these steps:

### Installation

Install the latest version of CyanPrint CLI using your preferred package manager:

### Create a project from a template

You can search for templates on the [cyanprint.dev](https://cyanprint.dev) registry.

```bash -cn
cyanprint create <template> <folder>
```

For example:

```bash -cn
cyanprint create cyan/new sample-project
```

The template `cyan/new` allows users to create new CyanPrint templates.

Follow and answer all the prompts, and your project will be ready to use!
