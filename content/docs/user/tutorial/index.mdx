# CyanPrint Tutorials

This section provides step-by-step tutorials to help you get started with CyanPrint. Follow these guides to learn how to install CyanPrint, use templates, and search the registry.

## Available Tutorials

- [Installation](/docs/user/tutorial/installation) - Learn how to install CyanPrint on different platforms
- [Use a Template](/docs/user/tutorial/use-template) - Create your first project using a CyanPrint template
- [Search the Registry](/docs/user/tutorial/search-registry) - Find templates in the CyanPrint registry

Each tutorial provides detailed instructions with examples to help you understand how to use CyanPrint effectively.

## Prerequisites

Before starting these tutorials, make sure you have:

- Basic knowledge of command-line interfaces
- A terminal or command prompt application
- Internet connection for downloading CyanPrint and templates

Ready to begin? Start with the [Installation](/docs/user/tutorial/installation) tutorial.
