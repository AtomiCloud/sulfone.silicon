# Installation

### Latest Version

Install the latest version of CyanPrint CLI.

<Tabs groupId="installer" queryString="installer">
  <TabItem value="nix-shell" label="Nix Shell">
    ```bash
    nix shell github:AtomiCloud/sulfone.iridium
    ```
  </TabItem>
  <TabItem value="nix-profile" label="Nix Profile">
    ```bash
    nix profile install github:AtomiCloud/sulfone.iridium
    ```
  </TabItem>
  <TabItem value="brew" label="Brew">
    ```bash
    bash brew install AtomiCloud/tap/cyanprint
    ```
  </TabItem>
  <TabItem value="scoop" label="Scoop">
    ```bash
    scoop bucket add atomi https://github.com/AtomiCloud/scoop-bucket.git
    scoop install atomi/cyanprint
    ```
  </TabItem>
  <TabItem value="apt" label="APT (Debian)">
    ```bash
    sudo apt install software-properties-common -y sudo
    apt-add-repository "deb [trusted=yes] https://apt.fury.io/AtomiCloud/ /" -y
    sudo apt update -y
    sudo apt install cyanprint
    ```
  </TabItem>
</Tabs>
