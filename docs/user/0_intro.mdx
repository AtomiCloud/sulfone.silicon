# Introduction

CyanPrint is a next-generation language-agnostic templating platform that helps you create projects from templates. It provides a powerful alternative to traditional project scaffolding tools with features like customizable templates, plugin support, and a central registry.

For a comprehensive understanding of CyanPrint's architecture and components, see the [Architecture Overview](/architecture).

## Quick Start

Quickly get started with CyanPrint by following these steps:

### Installation

Install the latest version of CyanPrint CLI using your preferred package manager:

<Tabs groupId="qs">
  <TabItem value="nix-shell" label="Nix Shell">
    ```bash
    nix shell github:AtomiCloud/sulfone.iridium
    ```
  </TabItem>
  <TabItem value="nix-profile" label="Nix Profile">
    ```bash
    nix profile install github:AtomiCloud/sulfone.iridium
    ```
  </TabItem>
  <TabItem value="brew" label="Brew">
    ```bash
    brew install AtomiCloud/tap/cyanprint
    ```
  </TabItem>
  <TabItem value="scoop" label="Scoop">
    ```bash
    scoop bucket add atomi https://github.com/AtomiCloud/scoop-bucket.git
    scoop install atomi/cyanprint
    ```
  </TabItem>
  <TabItem value="apt" label="APT (Debian)">
    ```bash
    sudo apt install software-properties-common -y
    sudo apt-add-repository "deb [trusted=yes] https://apt.fury.io/AtomiCloud/ /" -y
    sudo apt update -y
    sudo apt install cyanprint
    ```
  </TabItem>
  <TabItem value="yum" label="YUM (Redhat)">
    ```bash
    sudo sh -c 'echo "[fury]
    name=AtomiCloud Repo
    baseurl=https://yum.fury.io/atomicloud/
    enabled=1
    gpgcheck=0" > /etc/yum.repos.d/fury.repo'
    yum install cyanprint
    ```
  </TabItem>
</Tabs>

### Create a project from a template

You can search for templates on the [cyanprint.dev](https://cyanprint.dev) registry.

```bash
cyanprint create <template> <folder>
```

For example:

```bash
cyanprint create cyan/new sample-project
```

The template `cyan/new` allows users to create new CyanPrint templates.

Follow and answer all the prompts, and your project will be ready to use!
